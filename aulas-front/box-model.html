Box model do CSS

Definição

O Box model (modelo caixa) é um conceito fundamental no CSS que define como os elementos são dimensionados e posicionados na pagina

Nos exemplos abaixos, ambas as caixas possuem width:300px no CSS. mas, na pratica, suas dimensões finais são diferentes.
