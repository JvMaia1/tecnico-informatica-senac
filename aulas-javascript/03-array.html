Estrutura de dados: Array

Array (ou vetor) é uma estrutura usada para armazenar uma lista ordenada de dados em uma única variável ou constante.

Características principais:

  • Indexação zero: o primeiro elemento SEMPRE começa no índicie 0
  • Heterogêneos: você pode misturar diferentes tipos de dados (textos, números, objetos e outros arrays)
  • Tamanho dinâmico: o array aumenta ou diminui automaticamente conforme você adiciona ou remove itens.
  • Array dentro de array se torna uma matriz
